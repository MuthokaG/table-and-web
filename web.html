Web Programming

Learning Outcomes

  1. Define the benefits and power of CSS
  2. Write CSS-style rules.
  3. Articulate the big CSS concepts of inheritance, the cascade, specificity, rule order, and the box model.
  4. Understand the different page layouts.
  5. Use media queries for responsive design.

Required Readings

  • Chapter 4,5 (Internet and World Wide Web How To Program by Deitel & Deitel, 5th Ed)

Recommended Readings

  • Part II: CSS presentation (Chapter 11-19) (Learning Web Design: A Beginner's Guide to HTML, CSS, JavaScript, and Web Graphics 5th Edition)
  • What is the difference between fixed, fluid, adaptive, and responsive layouts and why should I care? Responsive Web Design - Media Queries
